### Hoe kan ik commentaar schrijven in JSX? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Commentaar komt hier */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* Het werkt ook voor commentaar 
  gesplitst over meerdere lijnen. */}
  Hello, {name}! 
</div>
```
